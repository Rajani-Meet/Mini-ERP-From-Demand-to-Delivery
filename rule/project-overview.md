# ShivERP

## Project Type
Manufacturing Mini ERP

## Tech Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- Shadcn UI
- Prisma ORM
- PostgreSQL
- Zustand
- React Hook Form
- Zod

## Modules
- Dashboard
- Products
- Sales Orders
- Purchase Orders
- Manufacturing Orders
- Bill Of Materials
- Inventory
- Audit Logs
- Users & Roles

## Business Model
MTO (Make To Order)
MTS (Make To Stock)

## Core Objective
Manage end-to-end manufacturing lifecycle:
Sales → Procurement → Manufacturing → Inventory → Delivery
